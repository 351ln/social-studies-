課前準備清單（可改）

  • 投影 / 大屏幕、學生分組座位安排
  • 案例素材：影片或貼文截圖（建議 2–3 則）
  • 學習單：觀點、證據、可信度、行動建議
  • 小白板 / 便利貼 / 計時器

小組評量規準（可改）

  • 能說出主張，並提出至少 1 個可信來源的證據
  • 能辨識「廣告 / 立場 / 可能偏誤」並做合理推論
  • 能提出可行的個人行動（不空泛）
  • 小組分工清楚、討論有紀錄

自學課程設計（給學生自己看也能懂）

目標：看懂 → 判斷 → 做選擇

自學路線圖（可改成你的版本）

  • 步驟 1：看一則素材（影片/貼文/新聞）並寫下「它在說什麼？」
  • 步驟 2：用 3 題檢核可信度：來源是誰？有數據嗎？有沒有反例？
  • 步驟 3：做一個選擇：你會買 / 不買 / 需要更多資訊？理由是什麼？
  • 步驟 4：提出一個行動：對自己、對同學、對家人可做的 1 件事

你也可以把自學做成「任務卡」：每完成一步就打勾 ✅

自學任務卡（範例）

  • ✅ 我能用一句話說出素材的主張
  • ✅ 我能找出至少一個來源並評估可信度
  • ✅ 我能寫下我的消費選擇與理由
  • ✅ 我能提出一個可行的行動
